# DevLog — краткое описание проекта

## Что это

Веб-приложение **«Дневник учёбы»**: пользователь ведёт учёт учебных сессий (дата, время начала/конца, перерывы в минутах, заметка «что делал»). Система считает чистое время занятий и показывает статистику на дашборде.

**Язык интерфейса:** русский.

---

## Стек

- **Бэкенд:** Node.js, Express, SQLite (better-sqlite3). Один файл БД `devlog.db`, таблицы: `sessions`, `week_notes`, `month_notes`.
- **Фронтенд:** ванильный HTML + CSS + JS (ES-модули, без фреймворков и бандлера). Код разбит на 5 модулей: `app.js`, `utils.js`, `api.js`, `storage.js`, `chart.js`.
- **Графики:** Chart.js 4.x (CDN), линейные диаграммы с кастомными тултипами и линией цели.
- **Хранение на клиенте:** localStorage (черновик формы, состояние таймера паузы, пороги графиков).

---

## Структура интерфейса

### Шапка (общая для всего приложения)

- Слева: заголовок «Дневник учёбы» + три вкладки навигации — **«Дневник»**, **«Дашборд»**, **«Настройки»**.
- Справа: при «Дневнике» — итоги недели/месяца; при «Дашборде» — «Всего за всё время: X ч Y мин».

### Вкладка «Дневник»

- **Цели на эту неделю** — список целей из предыдущей недели с чекбоксами (выполнена / не выполнена).
- **Форма «Новая сессия»** — поля: Дата, Начало, Конец, Перерывы + таймер паузы; textarea «Что делал»; кнопка «Добавить сессию». Кнопка очистки формы (✕) при наведении. Поддержка редактирования существующих сессий.
- **Таблица сессий** — переключатель «Текущая неделя / Текущий месяц», навигация по месяцам и неделям. Колонки: Дата, Время, Перерывы, Занимался, Что делал, кнопки редактирования и удаления.
- **Итоги и цели выбранной недели** — текстовое поле итогов, цели на следующую неделю.
- **Итоги месяца** — textarea в конце страницы.

### Вкладка «Дашборд»

- **Итоги месяца** — текстовый блок (скрыт, если не заполнен).
- **Часы по неделям** — линейный график (последние 8 недель), линия цели, кастомный тултип с информацией о часах, цели, отклонении, кол-ве сессий.
- **Часы по месяцам** — линейный график (последние 12 месяцев), аналогичный тултип.

### Вкладка «Настройки»

- Порог часов в неделю (по умолчанию 10).
- Порог часов в месяц (по умолчанию 40).
- Значения используются для линий цели на графиках.

## Визуальный стиль

- **Тёмная тема (GitHub-style):** фон `#0d1117`, карточки `#161b22`.
- **Текст:** светлый `#c9d1d9`, приглушённый `#8b949e`.
- **Акцент:** зелёный `#2ea043` (кнопки, итоги, точки графика).
- **Графики:** голубая линия `#5b9bd5`, жёлтый пунктир цели.
- **Шрифт:** системный (Segoe UI / system-ui).
- **Тултипы:** плавное появление/исчезание, плавное перемещение между точками.

---

## Ключевые сценарии

- Ввод/редактирование сессии с автоподстановкой даты и времени (Asia/Omsk).
- Таймер паузы: «Начать перерыв» → «Закончить перерыв», минуты добавляются в поле.
- Черновик формы сохраняется в localStorage, восстанавливается при перезагрузке.
- Удаление сессии с toast-уведомлением и возможностью отмены (5 сек).
- Переключение вкладок без перезагрузки (SPA), шапка адаптируется к активной вкладке.
- Недельные цели переносятся между неделями: цели из «Итогов недели» отображаются на следующей неделе как чеклист.

---

## Ограничения

- Одна страница (SPA): три вкладки, без маршрутизации.
- Ориентация на десктоп (max-width: 1000px).
- Без авторизации — однопользовательское приложение.
